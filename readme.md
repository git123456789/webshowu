# 秀站分类目录

## 简介

一个基于laravel的分类目录、导航程序！这里只是简单的完善了，后续还有很多强大的功能正在完善中……

秀文档还在积极的完善中……请大家耐心等待……

是lambq一个独立开发程序、在lambq学习laravel5.2的时候有感而来……

## 有哪些功能？
* 文章发布
* 网站收录
* 吊炸天的秀妹机器人，基于反向索引上一个外链接、自动收录。缩略图、数据采集、异步列队处理数据方式等
